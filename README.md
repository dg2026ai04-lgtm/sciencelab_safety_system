# sciencelab_safety_system
#  약품 실험실 스마트 안전 관리 시스템

## 프로젝트 소개
MQ2 가스 센서와 LED를 활용하여
실험실 내 유해가스 농도를 실시간으로
감지하고 단계별로 경고하는 시스템입니다.

## 팀원 및 역할
- 조윤서 (약학 지망): 안전 기준 설정
- 조현서 (화학생명공학 지망): 회로 구성, 코드 작성

## 경고 단계
| 단계 | LED | 센서값 |
|------|-----|--------|
| 안전 | 🟢 초록 | 0~300 |
| 주의 | 🟡 노랑 | 300~600 |
| 위험 | 🔴 빨강 | 600~800 |
| 긴급 | 🔴 점멸 | 800 이상 |
